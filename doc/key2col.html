NAME

key2col - generate a columnar cipher from an aristocraty key

SYNOPSIS

key2col filename

DESCRIPTION

key2col filename
key2col generates a columnar cipher from an aristocrat key. The input filename must be an aristocrat saved file from the ctool program. The output of this program is another ctool save file that can be loaded into ctool. This new save file is a complete columnar cipher with period 26.

By converting the aristocrat key to a columnar cipher, one may then change the key ordering in order to recover the K3/K4 mixed alphabets.
