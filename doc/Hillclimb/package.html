NAME

Hillclimb - Procedures for creating and using hill climbers.

SYNOPSIS

package require Hillclimb

VARIABLES

Hillclimb::neighborProc
This variable stores the name of the command that is used to generate 'nearby' keys in the key search space. Most neighborProc procedures generate neighbor keys by swapping elements of the current key.
